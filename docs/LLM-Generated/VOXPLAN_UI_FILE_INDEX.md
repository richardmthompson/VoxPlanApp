## USAGE NOTES

### For Developers
1. **Main Screen Changes** → Modify `/ui/main/MainScreen.kt` and `/ui/main/MainViewModel.kt`
2. **Navigation Changes** → Update `/navigation/VoxPlanScreen.kt` and `/navigation/VoxPlanNavHost.kt`
3. **New Colors** → Add to `/ui/constants/Colors.kt`
4. **New Strings** → Add to `/res/values/strings.xml`
5. **New Components** → Create in appropriate `/ui/*/` subdirectory

### For UI Designers
1. **Color Palette** → See `/ui/constants/Colors.kt` and `/ui/theme/Color.kt`
2. **Dimensions** → See `/ui/constants/DpValues.kt`
3. **Typography** → See `/ui/constants/TextStyles.kt` and `/ui/theme/Type.kt`
4. **Icons** → Located in `/res/drawable/` and `/res/mipmap-*/`

### For Testers
1. **Navigation Flow** → See VOXPLAN_UI_QUICK_REFERENCE.md (Section 13)
2. **Component Details** → See VOXPLAN_UI_ARCHITECTURE.md (Section 2)
3. **State Management** → See VOXPLAN_UI_ARCHITECTURE.md (Section 3)
